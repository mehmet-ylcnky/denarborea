DenArborea Examples
==================

This directory contains example files to demonstrate the file viewer functionality.

Files included:
- config.json: JSON configuration example
- data.csv: CSV data with employee information  
- docker-compose.yml: YAML service configuration
- Cargo.toml: TOML project configuration
- README.txt: This text file

Usage:
Use the --view flag to display file contents with syntax highlighting
and structured formatting based on file type.

Examples:
  denarborea --view config.json
  denarborea --view data.csv
  denarborea --view docker-compose.yml
